### Nodejs Hall Booking API

```
https://hall-booking-api-4lcj.onrender.com/ -> Get a list of all routes
https://hall-booking-api-4lcj.onrender.com/room -> Create a room
https://hall-booking-api-4lcj.onrender.com/rooms -> List of all rooms
https://hall-booking-api-4lcj.onrender.com/customer -> Create a customer
https://hall-booking-api-4lcj.onrender.com/book -> Book a room
https://hall-booking-api-4lcj.onrender.com/bookings -> List of all bookings
https://hall-booking-api-4lcj.onrender.com/bookings/:customerName/:roomName -> List of all bookings by {customerName} for {roomName}
```
